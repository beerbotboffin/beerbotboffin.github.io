Collection of my libraries 26 Jul, 2020

Whilst I like Fusion 360, I've burnt a couple of times with hosted software and hence I frequently design in Openscad.

What I try to do as often as possible is to create (or use a design by someone else, if available) modular designs so that they're reusable, and the common ones - things like boxes and cases for example I include in a library.
What I've done is to create a template (which I keep in Notepad ++) with a preconfigured format to which I've been adding a 'help' section so that I can remember what I've got in the library.

I've added a library to Github, (which you can find here and will be adding new ones whenever I create them.
